# eslint-config-rcm
ESLint config based on Airbnb config. Support TS, TSX, React.

## Installation

```console
$ npm install --save-dev eslint-config-rcm @typescript-eslint/eslint-plugin
```

## Usage

Set your eslint config to:

```json
{
  "extends": "rcm",
  "plugins": [
    "@typescript-eslint/eslint-plugin"
  ]
}
```
